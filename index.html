HTML Cheatsheet

Table tags

Tag Name Description
<table> Table The wrapper element to creat a table in HTML language.
<tr> Table Row Creates a row in the table that can be filled with information.
<td> Table Data Specify the data that goes inside within a table row.
<thead> Table Headings Sections the table to define headings in a table.
<tbody> Table body Section the table and indicates the rows that contain the table data.
<tfoot> Table foot Section the table into de bottom part of it, set of rows defining the footer of a table.

Table Attributes

Tag Name Description
<colspan> Column Span Indicates how many columns a td element should span
<rowspan> Row Span Indicates how many rows a td element should span

CSS Cheatsheet

Color Attributes

Attribute Name Description
color Color Sets the color of the element, it can be specify by name, RGB code or Hex code.
background-color Background Color Specify the background color of an element it can be set by name, RGB code or Hex code.

Syntax for rgb, hsl & hsla

Syntax Description
rgb(n,n,n) The rgb has to be set by three parameters(red, green and blue) that defines the intesity and can be from 0 to 255.
hsl This syntax means: HUE that can be set in a number between 0 to 360,Saturation that is define in percentage, Lightness that also is define in percentage.
hsla This syntax is the same as the previous one, just with an add that is a(alpha) this value defines the opacity and is set from 0 to 1.
